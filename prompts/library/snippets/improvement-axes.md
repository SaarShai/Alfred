# Improvement axes

<!-- promptkit:start -->
Improvement axes:
- More efficient, more productive, faster, safer, more reliable, easier to maintain, better output, higher quality, lower cost, better retrieval, better memory, better delegation, better verification, better UX, or improved in any meaningful way.
- Prioritize candidates with clear leverage for the current project over merely interesting items.
