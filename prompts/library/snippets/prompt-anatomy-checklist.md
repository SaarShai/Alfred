# Prompt anatomy checklist

<!-- promptkit:start -->
Prompt construction checklist:
- State the task once, clearly.
- Delimit runtime inputs.
- Include only constraints that change the output.
- Put the output format near the end.
- Use examples only when the format or judgment boundary is hard to infer.
