# STABILIZERS

## CONNECTOR STABILIZERS

* LAYERS = front layer + back layer

* LENGTH = 1095mm

* STABILIZER SLOTS 
  - WIDTH= the total width of a connector stabilizer + 0.4mm clearance (default: 12mm + 12mm + 0.4mm = 24.4mm)
  - short: LENGTH = 85mm
  - long: LENGTH = 462.5mm

* Ref images


Types: bottom stabilizer (small, at base); connector stabilizer (connecting top subpanels with bottom subpanels).
3.2.1 Layers: 
3 layers corresponding to the 3 layers of panels in a set, or 2 layers corresponding to front and back layers of panels.
3.2.2 Slots on stabilizers (short / long)
“Long slot on stabilizer” — at BASE, splits bottom into 2 splayed feet. Mates with bottom subpanel's `top stabilizer slot`.
Length = 462.5 mm.
Width = total panel thickness + 0.6 mm clearance. Example: for a 3-layer panel at 12 mm each layer: 36.6 mm. 

“Short slot on stabilizer” — at TOP end. Mates with top subpanel's `bottom stabilizer slot`.
Length = 85 mm
Width = total stabilizer thickness + 0.6 mm clearance; same as ‘long slot on stabilizer’.

“Accommodation area” - a non-cutting area that continues ‘into’ the stabilizer (length-wise) from the edge of the slot. This area accommodates a corresponding stabilizer slot of the same length in the a panel. Width and length of accommodation area are the same as the corresponding stabilizer slot.

3.2.3 Connector stabilizers

Default height: 1095 mm.

Standard door panel pairs with 2 connector stabilizers (left + right); 4 panel-side slots (2 in top subpanel + 1 in each bottom).

Standard narrow panel pairs with 1 connector stabilizer; 2 panel-side slots (1 in top subpanel, 1 in bottom).

Finger holes: typically 2 sets / 4 holes total — one per side.
